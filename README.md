# git-playground

## 概要
### Git学習用🧐

## 使い方

### 1. issueを立てる📝
issueの例：
> ## fooコンポーネントを追加 #1
> ### 内容
> hogefugaするコンポーネントを追加する。
> ### 目的
> なんやかんやに必要なため。

### 2. ローカルでブランチを切る🌿
ブランチ名の例： _**feature/#1_add_foo_component**_

### 3. プルリクエストを作成する🙇‍♀️
PRの例：
> ## add: fooコンポーネントを追加
> ## issue
> #1(issue番号)

### 4. マージする🎉🎉
マージコメントの例：
> ## add: fooコンポーネントを追加 (#2)(PR番号)
